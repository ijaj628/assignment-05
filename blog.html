What is the Document Object Model(DOM)?

Document Object Model is a programming method that use for programming interface for web documents.It give the structure of a webpage as a tree of objects.

How do you select an element from the DOM?

We can select an element form DOM using the attributes 'getElementById', 'getElementsByClassName', 'getElementsByTagName', 'querySelector', 'querySelectorAll' etc.

What is event delegation in the context of the DOM,and why is it useful?

It is use for to the page to make sure that page dosn't relode.

How do you manipulate an element's attributes and style using the DOM?

DOM is straightforward with JavaScript.So we can access DOM element using 'getElementById', 'querySelector', 'getElementsByClassName', Or changing attributes by using 'setAttribute', 'removeAttribute', and using style property.
